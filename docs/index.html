Belajar Skills AI

Selamat datang di dokumentasi Belajar Skills AI. Proyek ini mendemonstrasikan bagaimana membangun AI Agent prosedural yang mampu menjalankan fungsi-fungsi spesifik (skills) secara otomatis dan logis.

Apa itu Skills AI?

Skills AI atau Agentic Workflow adalah kemampuan model bahasa besar (LLM) untuk berinteraksi dengan dunia nyata melalui kode. Agent tidak hanya sekadar bicara, tapi bisa mengambil tindakan seperti mengecek stok gudang, menghitung data, hingga mengirim email secara mandiri.

Cara Setup & Instalasi

Ikuti langkah-langkah berikut untuk menjalankan agent di komputer lokal Anda:

  1. Pastikan Anda memiliki Python 3.8+ terinstall.
  2. Clone atau unduh folder belajar_skills_ai.
  3. Buka terminal dan instal dependensi:
    pip install -r requirements.txt
  4. Atur API Key DeepSeek Anda:
    set DEEPSEEK_API_KEY=your_key_here

Cara Penggunaan

Menjalankan agent sangatlah mudah. Cukup panggil script utama:

python app.py

Contoh Prompt

Anda bisa mencoba memberikan instruksi kompleks seperti:

"Saya ingin pesan keyboard, email saya adi@example.com."

Agent akan otomatis:
1. Memeriksa stok keyboard.
2. Menghitung harga.
3. Mengirim konfirmasi ke email yang disebutkan.

Cara Mempelajari

Ingin mendalami bagaimana logic agent bekerja? Fokuslah pada 3 bagian kunci di file app.py:

1. Tool Definition

Lihat bagaimana kita mendefinisikan fungsi Python dan skema JSON-nya untuk "mengajar" AI cara memanggilnya.

2. The Agent Loop

Perhatikan struktur while True yang menangani tool_calls secara berulang hingga tugas tuntas.

3. System Message

Eksperimenlah dengan mengubah instruksi sistem untuk mengubah perilaku prosedural sang Agent.
